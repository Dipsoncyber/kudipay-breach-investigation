KudiPay Breach Investigation

Cybersecurity capstone project simulating a breach investigation of a fictional 
microfinance bank ("KudiPay"). Completed as part of a cybersecurity bootcamp, 
covering six investigation tracks: network analysis, SOC log triage, 
penetration testing, digital forensics, AWS security, and GRC.

This is a training exercise, not an investigation of a real company or incident.
